## 언리얼 엔진 기초

## 언리얼 vs 유니티

- 언리얼
  - 태생이 있고 확장시켜나감(상속 패턴 구조)
  - 삭제, 이름 변경 어려움
    - 언리얼 엔진 끄기 -> 파일 관리자에서 소스 파일 삭제
- 유니티

  - 빈 깡통에서 부품을 맞춰 나감(컴포넌트 패턴 구조)
  - 삭제, 이름 변경 쉬움

### 실습: 의자를 하나 만들어보자

1. Actor 생성
2. Mesh Component 추가
   - 코드
3. 의자 Mesh 파일 경로 복사(`Ctrl+C` or `메시 오른쪽 클릭+파일 경로 복사`)
4. 해당 Actor에 가져온 Mesh 설정
